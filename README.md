# ⏱️ Stopwatch

A minimal and responsive stopwatch application built with HTML, CSS, and JavaScript.

## 🔗 Live Demo
[View Stopwatch](https://ioana2803.github.io/counter/)

## ✨ Features
- Tracks elapsed time in HH:MM:SS format
- Start, Pause, and Reset controls
- Dark display on a vibrant cyan-to-purple gradient background
- Clean minimal UI

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (Vanilla)
